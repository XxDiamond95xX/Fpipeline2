# ⚡ Ebook Agent Pipeline

Pipeline de 6 agents IA pour créer et publier des ebooks automatiquement.

## 🚀 Déploiement en 5 minutes sur Vercel

### Étape 1 — Créer le dépôt GitHub
1. Va sur [github.com](https://github.com) → **New repository**
2. Nomme-le `ebook-pipeline`
3. Laisse-le public → clique **Create repository**

### Étape 2 — Uploader les fichiers
Sur la page du repo GitHub :
1. Clique **uploading an existing file**
2. Glisse TOUS les fichiers de ce dossier (sauf `node_modules`)
3. Clique **Commit changes**

### Étape 3 — Déployer sur Vercel
1. Va sur [vercel.com](https://vercel.com) → **Sign up with GitHub**
2. Clique **New Project**
3. Importe ton repo `ebook-pipeline`
4. Clique **Deploy** (Vercel détecte Vite automatiquement)
5. ✅ Tu obtiens un lien public !

## 🔑 Utilisation

1. Entre ta **Claude API Key** → [console.anthropic.com](https://console.anthropic.com/settings/keys)
2. Entre ta **niche** d'ebook
3. Clique **▶ LANCER**
4. Après génération, entre ton **token Gumroad** pour publier
